# 连接应用

应用可以让智能体连接外部系统。比如连接企业微信、飞书、钉钉、文档系统、代码仓库、数据库、MCP 服务等。

简单理解：应用是智能体的“连接口”。没有应用时，智能体主要依赖你在聊天里提供的信息；连接应用后，它就可以在授权范围内访问外部系统或接收外部渠道消息。

## 1. 应用能做什么

应用通常用于：

- 从外部系统读取信息。
- 把结果发送到 IM 或业务系统。
- 调用企业内部工具。
- 连接 MCP 服务。
- 让智能体在其他渠道中被使用。

简单理解：应用负责“连到哪里”。

## 先分清应用和渠道

在日常使用里，你可以这样区分：

- **应用**：连接一个能力或系统，例如文档、CRM、数据库、代码仓库、MCP 服务。
- **渠道**：让你从哪里和智能体聊天，例如企业微信、飞书、钉钉、个人微信。

有些渠道也会以应用或 IM 工具的形式出现。你不用纠结分类，按页面提示完成授权或凭证配置即可。

## 2. 创建智能体时连接应用

在创建向导最后一步切换到 `Apps`。

你会看到应用列表，通常分为：

- **已配置应用**：可以直接启用。
- **未配置应用**：还需要补充凭证或授权。

打开需要的应用即可。

![创建时选择 Apps](/images/docs/create-agent-apps.jpg)
## 3. 创建后连接应用

如果智能体已经创建完成：

1. 打开智能体详情页。
2. 点击顶部 `Apps` 标签。
3. 搜索或找到要连接的应用。
4. 打开开关。
5. 如果提示需要配置，点击 `Configure`。

![详情页 Apps 标签](/images/docs/app-detail.jpg)
## 4. 配置应用凭证

不同应用需要填写的内容不同，常见字段包括：

- Endpoint URL。
- Auth Token。
- API Key。
- OAuth 授权。
- 应用 ID。
- 企业 ID。
- 回调地址。

配置步骤：

1. 在应用行点击操作菜单。
2. 点击 `Configure`。
3. 按表单填写凭证。
4. 保存。
5. 等待状态同步。

## 5. 示例：连接钉钉文档 MCP

下面用“钉钉文档”作为例子，说明一次完整的应用对接流程。

### 5.1 在钉钉 MCP 广场复制地址

1. 打开 [钉钉 MCP 广场](https://aihub.dingtalk.com/#/mcp)。
2. 搜索或找到“钉钉文档”。
3. 在右侧配置区域找到 `StreamableHttp URL`。
4. 复制这个 URL。

这个 URL 是智能体访问钉钉文档能力的连接地址。

![钉钉 MCP 广场](/images/docs/dingtalk-docs-market.jpg)
### 5.2 在 ZStack 虾厂 中配置钉钉文档应用

回到 ZStack 虾厂：

1. 打开智能体详情页。
2. 进入 `Apps` 标签。
3. 找到“钉钉文档”应用。
4. 点击 `Configure`。
5. 将刚才复制的 `StreamableHttp URL` 粘贴到对应参数中。
6. 选择要连接到哪个智能体。
7. 保存配置。

保存后，系统会把钉钉文档能力绑定到你选择的智能体上。

### 5.3 在智能体中测试

配置成功后，进入该智能体的 `Chat` 页面，直接提问即可。

你可以尝试：

- “当前有哪些钉钉文档？”
- “帮我列出最近的文档。”
- “读取某个文档的内容。”
- “总结这个文档的重点。”

如果配置正确，智能体会通过钉钉文档 MCP 读取可访问的文档信息，并基于文档内容回答。

![智能体读取钉钉文档](/images/docs/dingtalk-docs-chat.jpg)
### 5.4 如果读取失败

优先检查：

- `StreamableHttp URL` 是否复制完整。
- 钉钉文档 MCP 是否可用。
- ZStack 虾厂 中是否保存到了正确的“钉钉文档”应用参数。
- 是否选择并绑定到了正确的智能体。
- 当前钉钉账号是否有目标文档的访问权限。
- 应用状态是否显示 `Active`，或是否仍处于 `Needs Configuration` / `Error`。

## 6. 查看应用状态

应用列表中可能出现以下状态：

- `Active`：已经生效。
- `Pending Sync`：正在同步到智能体运行时。
- `Needs Configuration`：缺少必要配置。
- `Blocked`：被工作空间策略或权限阻止。
- `Error`：配置失败或运行异常。

如果看到 `Needs Configuration`，先补全凭证。

如果看到 `Blocked`，通常需要联系管理员。

## 7. 连接 IM 渠道

如果你希望在微信、企业微信、飞书、钉钉等渠道里使用智能体，可以在智能体详情页的 `Information` 页面找到 `Channel` 区块。

操作步骤：

1. 点击 `Add IM tool` 或 `Add`。
2. 选择渠道类型。
3. 填写或绑定渠道凭证。
4. 保存。
5. 等待健康检查。

![Channel 区块](/images/docs/channel-config.jpg)
更完整的渠道配置说明请查看 [渠道对接](/docs/channel-integration)。

## 新手建议

- 先连接一个最必要的应用。
- 应用凭证尽量由系统管理员提供。
- 连接后先用一个简单任务测试是否可用。
- 如果应用涉及敏感数据，确认工作空间策略和模型服务是否合规。
