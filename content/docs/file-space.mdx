# 文件空间

文件空间用于存放智能体在工作过程中产出的各种文件。你可以在这里查看文件列表、确认文件内容，并按需下载到本地继续使用。

常见文件包括：

- 智能体生成的报告、方案、纪要或总结。
- 对话中导出的文档。
- 任务执行过程中产生的表格、图片或附件。
- 智能体读取、整理或转换后的业务资料。

![文件空间](/images/docs/files.jpg)

## 1. 进入文件空间

打开智能体详情页后，进入 `Workspace` 或文件相关入口。

如果页面顶部有多个标签，文件空间通常和 `Chat`、`Information`、`Apps`、`Skills` 等入口并列。

## 2. 查看文件

文件空间中通常会展示：

- 文件名称。
- 文件类型。
- 创建或更新时间。
- 文件大小。
- 关联的对话、任务或来源。

看到不熟悉的文件时，可以先打开预览，确认它是否是你需要的结果文件。

## 3. 下载文件

需要保存文件时，点击文件对应的下载操作。

下载前建议确认：

- 文件是否是最新版本。
- 文件内容是否已经检查过。
- 文件是否包含敏感信息，是否可以发送给外部人员。

## 4. 使用建议

- 让智能体生成文件后，可以到文件空间确认是否产出成功。
- 如果一次任务会生成多个文件，先看文件名和更新时间，再下载需要的版本。
- 对重要文件，建议下载后按团队规范归档。
- 如果看不到文件，先确认智能体任务是否已经完成，或刷新文件空间。
